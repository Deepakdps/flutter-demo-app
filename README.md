# Flutter-Demo-App

This is an Cross Platform Mobile Application Using Flutter provided by Google

## Sections

SIDE DRAWER, LISTING, ROUTING, DEMO CHAT SECTION

## Installation

```
git clone https://github.com/Paulsonp/flutter-demo-app.git
cd flutter-demo-app
flutter packages get
flutter run
```
